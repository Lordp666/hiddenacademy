Programme de la formation

Module 1 : Reconnaissance

  • Découverte de services avec Nmap
  • Analyse de ports et bannières
  • Énumération web avec outils dédiés

Module 2 : Exploitation Web

  • Injection SQL / XSS basique
  • Bypass d’authentification
  • Introduction à Burp Suite

Module 3 : Capture de flags

  • Exploitation de failles simples
  • Escalade de privilèges basique
  • Bonnes pratiques en CTF

Ce que vous saurez faire après cette formation

  • Identifier les failles de base d’un site
  • Utiliser Nmap et Burp Suite efficacement
  • Exploiter des CTF simples en autonomie
  • Lire des write-ups techniques
  • Rejoindre une communauté CTF
